### selene 希腊神话-塞勒涅：商城小程序

### 安装vscode扩展插件
```
1、minapp
2、Live Sass Compiler

在.vscode -> settings.json添加

{
    "liveSassCompile.settings.formats": [
        {
            "format": "expanded",
            "extensionName": ".wxss",
            "savePath": null
        }
    ],
    "liveServer.settings.donotShowInfoMsg": true
}
```

### 在根目录添加config.js文件，内容为configSample.js的内容
